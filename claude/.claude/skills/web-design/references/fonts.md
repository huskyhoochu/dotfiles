## Recommended Pairings

### Neobrutalism
```
Heading: Space Grotesk (600, 700)  OR  Archivo Black (400)
Body: DM Sans (400, 500)
Accent: Space Mono (400, 700)
Korean: Noto Sans KR (400, 700, 900)
```

### Glassmorphism
```
Heading: Plus Jakarta Sans (600, 700)
Body: Plus Jakarta Sans (400, 500)
```

## Google Fonts CDN Template

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=HEADING:wght@WEIGHTS&family=BODY:wght@WEIGHTS&family=ACCENT:wght@WEIGHTS&family=Noto+Sans+KR:wght@400;500;700&display=swap" rel="stylesheet">
```

Replace HEADING, BODY, ACCENT, WEIGHTS with chosen fonts and weights.
