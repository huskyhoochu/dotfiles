# Brief Template (Game Research)

Use this structure for quick overviews (`--brief` mode). Fill all sections in **Korean**.

```markdown
# {topic} — 게임 브리프

> {date} | {n}개 소스 참조

## 요약
{synthesized_summary_from_initial_discovery}

## 핵심 포인트
- {point_1}
- {point_2}
- {point_3}
- ...

## 주요 출처
- [{title_1}]({url_1})
- [{title_2}]({url_2})
- ...
```

## Field Guidelines

- **요약**: 2-3 paragraphs synthesizing the Perplexity overview and Brave search results
- **핵심 포인트**: 5-8 bullet points covering the most important findings. For game titles, include review scores if available from discovery. For esports, include recent results.
- **주요 출처**: Top 5-8 most relevant sources with clickable links
- Keep the entire brief under ~500 words
